Description
===========

The ``debops.snappymail`` Ansible role manages `SnappyMail`__, an IMAP Web
client based on `RainLoop`__, but modernized to use current PHP versions, to
address unresolved security issues and to work well with current browsers
and mobile devices.

.. __: https://snappymail.eu/
.. __: https://rainloop.net/
